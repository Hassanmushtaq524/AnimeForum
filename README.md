### Installation


Clone the repository
```bash
git clone https://github.com/Hassanmushtaq524/TodoGame.git
```

Install the packages
```bash
cd /frontend/
npm i
cd ..
cd /backend/
npm i
```


### Usage

Frontend and Backend are run by a single docker-compose file:

```bash
docker-compose up --build
```
